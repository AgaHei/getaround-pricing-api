# Data Files

This directory contains the datasets used for the Getaround pricing analysis project.

## Files:
- `cleaned_getaround_data.csv`: Preprocessed dataset ready for analysis and modeling
- `get_around_delay_analysis.xlsx`: Analysis of rental delays and checkout behaviors  
- `get_around_pricing_project.csv`: Original dataset for pricing optimization

## Data Sources:
The data comes from Getaround's rental platform, containing information about car rentals, features, and pricing.
