🚗 Getaround Pricing API

This API predicts the optimal rental price per day for a car listed on Getaround, based on its features.

🔮 POST /predict

Input: JSON object with the following fields:

{
  "mileage": 12000,
  "engine_power": 110,
  "private_parking_available": true,
  "has_gps": true,
  "has_air_conditioning": true,
  "automatic_car": true,
  "has_getaround_connect": true,
  "has_speed_regulator": true,
  "winter_tires": true,
  "model_key": "Toyota",
  "fuel": "petrol",
  "paint_color": "black",
  "car_type": "suv"
}

Output: JSON object with predicted price:

{
  "prediction": [127.3]
}

Example (curl):

curl -X POST http://your-url.com/predict \
-H "Content-Type: application/json" \
-d '{ ... }'
